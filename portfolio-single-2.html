RentEase Application

Project Date August 9th, 2024
Platform Android App
View GitHub Repo

RentEase is a fully-featured Android car rental application developed using Java and Firebase, designed with two dedicated portals: User Portal and Admin Portal. It provides a seamless experience for users to browse, book, and manage rental cars while giving administrators complete control over car listings and user management. It offers a scalable and efficient solution for car rental businesses looking to automate their operations.

Key Features:

  • Secure Authentication: Login/register with Firebase Authentication.
  • Firebase Integration: Uses Firebase Firestore for real-time database management.
  • User Management: View, activate, deactivate, or remove users from the system.
  • Car Management: Add, update, or remove rental cars from the inventory.

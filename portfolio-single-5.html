Currency Converter

Project Date January 9th, 2025
Platform Android / Java
Integration Live Exchange APIs
View GitHub Repo

The Currency Converter is an Android application built using Java, providing real-time currency exchange rate conversions. It's fast, accurate, and user-friendly, ideal for travelers, businesses, and finance enthusiasts.

Key Features:

  • Fetches live exchange rates from reliable APIs (ExchangeRate-API, Open Exchange Rates).
  • Supports conversion between multiple currencies with up-to-date rates.
  • Allows users to select over 150 global currencies.
  • Displays converted amounts in real-time as users input values.
  • Built using Material Design for intuitive experience.
  • Simple input fields, dropdown selection, and instant results.
